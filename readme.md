# 📊 Sales Data Analysis using SQL

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a sales dataset using SQL.
The goal is to analyze customer behavior, product performance, and revenue distribution to generate meaningful business insights.

---

## 🗂️ Dataset Description

The analysis is based on three datasets:

* **Customers** – Contains demographic details such as name, gender, country, and birthdate
* **Products** – Includes product categories, subcategories, and cost information
* **Orders** – Transaction-level data including sales amount, quantity, and order dates

---

* PostgreSQL (pgAdmin)
* SQL

## 🔍 Analysis Workflow

### 1. Data Setup

* Created tables for customers, products, and orders
* Imported CSV files using SQL `COPY` command

### 2. Data Exploration

* Identified unique countries and product categories
* Analyzed sales date range
* Evaluated customer age distribution

### 3. Key Metrics

* Total Sales
* Total Quantity Sold
* Average Selling Price
* Total Orders
* Total Customers

### 4. Magnitude Analysis

* Customer distribution by country and gender
* Product distribution by category
* Revenue generated by each category
* Revenue generated by each customer

---

## 📈 Key Insights

* A small number of product categories contribute the majority of total revenue
* Customer distribution varies significantly across countries
* Revenue is concentrated among a small group of high-value customers
* Some product categories have higher costs but lower revenue contribution
