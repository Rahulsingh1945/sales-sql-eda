# 📊 Sales & Customer Analysis using SQL

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset using SQL.
It analyzes **customer behavior, product performance, and revenue distribution**, and builds a **customer-level report for deeper insights**.

---

## 🎯 Objective

* Understand customer purchasing behavior
* Identify high-value customers
* Analyze product and revenue performance
* Detect potential churn using recency

---

## 🗂️ Dataset Description

The analysis is based on three datasets:

* **Customers** → demographic details (name, gender, country, birthdate)
* **Products** → categories, subcategories, cost
* **Orders (Sales)** → transaction data (sales, quantity, dates)

---

## 🔍 Analysis Workflow

### 1. Data Setup

* Created tables for customers, products, and sales
* Imported CSV files using SQL

### 2. Data Exploration

* Identified unique countries and product categories
* Analyzed sales date range
* Evaluated customer age distribution

### 3. Key Metrics

* Total Sales
* Total Quantity Sold
* Average Selling Price
* Total Orders
* Total Customers

### 4. Magnitude Analysis

* Customer distribution by country and gender
* Product distribution by category
* Revenue generated by category
* Revenue generated by customers

---

## 📊 Customer Report (New Feature)

Built a **customer-level analytical report** using SQL:

### Includes:

* Total Orders, Sales, Quantity
* Total Products Purchased
* Customer Lifespan

### Segmentation:

* **VIP** → High-value, long-term customers
* **Regular** → Moderate engagement
* **New** → Recently acquired customers

### KPIs:

* **Recency** → Months since last purchase
* **Average Order Value (AOV)**
* **Average Monthly Spend**

---

## 📈 Key Insights

* A small number of customers contribute a large portion of total revenue
* Revenue is concentrated in specific product categories
* Customer distribution varies across countries
* High recency customers indicate potential churn risk
* Some categories have high cost but lower revenue contribution
